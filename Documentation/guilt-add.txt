guilt-add(1)
============

NAME
----
guilt-add - Add a file to git and guilt

SYNOPSIS
--------
include::usage-guilt-add.txt[]

DESCRIPTION
-----------
Adds the files specified to git using git-add making it available to guilt.

OPTIONS
-------
<files>::
	Files to add

EXAMPLES
--------

Create and add a new file example.c

	$ touch example.c
	$ guilt-add example.c
